## Current Logo

The current VPC logo uses an industrial construction scene with a crane, building, excavator, dark blue border, orange sunrise background, and a strong `VPC` monogram. The website uses the official PNG logo asset at:

```text
website/assets/logo-vpc.png
```

Keep this PNG as the primary logo reference across the website, including the navbar, footer, browser tab favicon, and Apple touch icon.

## WhatsApp Business Conversation

The homepage includes a WhatsApp enquiry bot panel. Before launch, update the business WhatsApp number in `website/index.html`:

```html
<div class="whatsapp-bot" data-whatsapp-number="919999999999">
```

Use the full country code and number without `+`, spaces, or hyphens.
